# Фибоначчи

## Функция должна:
1. Принимать один числовой параметр
2. Возвращать элемент последовательности Фибоначчи

Пример:
```javascript
fibonacci(1); // 0
fibonacci(2); // 1
fibonacci(11); // 55
```

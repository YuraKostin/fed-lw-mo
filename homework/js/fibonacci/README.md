# Фибоначчи

## Функция должна:
1. Принимать один числовой параметр
2. Возвращать элемент последовательности Фибоначчи

Пример:
```javascript
fibonacci(0); // 0
fibonacci(1); // 1
fibonacci(10); // 55
```

при корректном входящем аргументе?
